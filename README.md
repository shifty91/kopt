# Kopt #

## About ##

`kopt` is an argument parsing library written in C++ for C++.

## Usage ##

See examples in `examples` directory.

## Build ##

    $ mkdir build
    $ cd build
    $ cmake -DCMAKE_INSTALL_PREFIX=/usr -DCMAKE_BUILD_TYPE=Release ..
    $ make -j8
    $ sudo make install

## Dependencies ##

- Modern Compiler with CPP 17 Support
- Getopt
